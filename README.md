# ДЗ React Project з використанням хука use()

## Опис проекту

Цей проект демонструє використання нового React хука `use()` для роботи з Promise в компонентах. Проект побудований на базі React + Vite і включає компонент `MessageComponent`, який асинхронно завантажує дані з симульованого серверного запиту.

### Основні можливості:
- Використання хука `use()` для отримання даних з Promise
- Інтеграція з React Suspense для відображення стану завантаження
- Симуляція асинхронного запиту до сервера з затримкою
- Сучасний стек: React 18+ та Vite

## Встановлення

### Передумови
Переконайтеся, що у вас встановлено:
- Node.js (версія 18 або вище)
- npm або yarn

### Кроки встановлення

1. Клонуйте репозиторій або завантажте файли проекту

2. Встановіть залежності:
```bash
npm install
```

або якщо використовуєте yarn:
```bash
yarn install
```

## Запуск проекту

### Режим розробки

Запустіть сервер розробки:
```bash
npm run dev
```

або:
```bash
yarn dev
```

Після запуску відкрийте браузер за адресою, яку покаже Vite (зазвичай `http://localhost:5173`)

### Збірка для продакшену

Створіть оптимізовану збірку:
```bash
npm run build
```

### Попередній перегляд продакшн-збірки

```bash
npm run preview
```

## Як це працює

Компонент `MessageComponent` використовує хук `use()` для отримання даних з Promise. Promise створюється функцією `fetchMessage()`, яка симулює асинхронний запит із затримкою 2 секунди. Компонент обгорнутий у `Suspense`, який показує індикатор завантаження до отримання даних.
